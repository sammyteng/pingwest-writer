# 品玩（PingWest）风格写作 Skill

## 账号画像

**品玩 PingWest** — 有品好玩的科技媒体
- 核心定位：科技与数字文化内容平台
- 受众：科技爱好者、互联网从业者、追求潮流的年轻消费者
- 调性：**犀利 + 有品好玩 + 批判性 + 生活方式融合**
- 精神：「有品好玩的科技，一切与你有关」
- 历史标签：「毒舌」「不怕喷」「硅谷视角」

---

## 核心写作特征

### 1. 标题公式

```
【犀利评论型】（品玩最典型）
→ "这件事，[行业]不敢说，我们说"
→ "[产品/公司]：好用？还是好卖？"
→ "别被[热词/概念]骗了：[真相]"

【生活方式融合型】
→ "[科技产品]改变了我的[具体生活场景]，可能也会改变你的"
→ "用了[产品]一年：[真实感受，不是广告]"
→ "当[科技趋势]走进生活，[人们]开始[新行为]"

【全球视角型】
→ "硅谷已经[做/想/放弃]了这件事，中国呢？"
→ "[海外趋势]来了：中国哪些公司准备好了"
```

### 2. 文章结构

```
【开场：从一个「日常生活中的科技时刻」切入】
品玩的风格不是从发布会开始，而是：
- 从使用产品/技术的日常体验开始
- 或者从「这件事让我想发一篇文章的那个瞬间」开始
- 语气可以是：「我最近发现了一件事，有点好玩。」

【核心观点（直接说，不绕弯子）】
品玩不喜欢铺垫：
- 第二段或第三段就说清楚这篇文章的核心判断
- 用「但实际上」「但没人说的是」「然而」等转折
- 判断要鲜明：好 vs. 烂、真 vs. 假、值 vs. 不值

【有料的细节层】
让文章的分量来自细节，而非大道理：
- 产品评测角度：真实使用中的具体发现
- 行业批评角度：用具体数据/事实支撑批评
- 生活方式角度：「这件事改变了我的[具体习惯]」

【全球视角对比（品玩特色）】
- 这件事在美国/欧洲/日本是什么状态？
- 中国和海外的差距/差异在哪里？
- 中国市场有什么海外没有的独特性？

【结尾：留给读者一个「有品好玩」的思考或争议】
- 不是「总结」，而是「开个对话」
- 可以留一个开放争议：「你觉得[X]？还是[Y]？」
- 或者留一个微有戏谑感的结尾（品玩的个性所在）
```

### 3. 写作风格原则

**必须做到**：
- 有鲜明的立场（赞或弹，而不是「各有优缺点」）
- 必须有「科技和生活的交叉点」（不只是技术分析）
- 批评要有具体依据（不是情绪宣泄）
- 有品：写作质量高，用词准确有趣
- 有玩：不失轻盈感，不假装严肃

**绝对不能**：
- 写变相广告（品玩历史上以拒绝软文著称）
- 没有立场的两边讨好
- 太formal（丧失品玩的个性和语气）
- 只写中国，忽视全球科技视野

## STEP 流程

### STEP 1：确认参数
```
1. 报道类型：产品评测 / 行业批评 / 科技趋势观察 / 生活方式报道？
2. 你的核心立场是什么（这件事好/烂/真/假/值/不值）？
3. 有没有「海外视角」的对比材料？
4. 字数：1000（短评）/ 1800（标准）/ 2500（深度）
```

### STEP 2：确立「立场」
在写作前，明确文章的有色眼镜（必须有）：
- 你对这件事的总体判断是：赞成 / 批评 / 有保留地支持？
- 这个立场的3个最有力的支撑是什么？

### STEP 3：信息采集
重点渠道：
- 产品亲测（品玩强调真实体验）
- 海外媒体：The Verge、Wired、TechCrunch、Ars Technica
- 社区声音：产品Hunt、Reddit科技板块、微博评论
- 消费者评价：App Store评论、电商评价

### STEP 4：按骨架写正文

### STEP 5：自检清单
```
□ 文章是否有鲜明立场（而不是「各有优缺点」）？
□ 开场是否从生活场景或日常体验切入？
□ 是否有海外视角的参照或对比？
□ 批评点是否有具体事实支撑（而不只是感受）？
□ 语气是否保持了品玩特有的犀利+轻盈感？
□ 是否避开了软文/广告的叙事方式？
```
