# 🎮 品玩（PingWest）Writer Skill

> 让 AI 以品玩「有品好玩+批判犀利」风格写科技生活方式内容的可复用 Skill。

## 简介

本 Skill 基于对品玩 PingWest 数百篇科技观察文章的结构化解析，将品玩「犀利立场+科技即生活+全球视角」的独特风格标准化为 AI 写作指令。

## 文件结构

```
pingwest-writer/
├── SKILL.md                          # 主技能文件
└── resources/
    ├── playbook_product_review.md    # 骨架一：产品犀利评测
    ├── playbook_industry_critique.md # 骨架二：行业批判观察
    ├── playbook_tech_lifestyle.md    # 骨架三：科技生活方式
    ├── playbook_global_compare.md    # 骨架四：全球科技对比
    └── playbook_trend_observe.md     # 骨架五：趋势犀利观察
```

## 品玩核心写作风格

| 维度 | 特征 |
|------|------|
| 立场 | 鲜明，不做两边讨好 |
| 语气 | 犀利但有品，不粗暴 |
| 视角 | 科技与生活融合，有全球参照 |
| 特色 | 拒绝软文，敢说真相 |
| 禁忌 | 没立场的中间路线、广告化叙事 |

## 核心写作哲学

- **有品**：内容质量高，文字讲究，洞察有深度
- **好玩**：不失轻盈感，科技融入真实生活场景
- **犀利**：对炒作和虚假说真话，不怕得罪人
- **全球视野**：以国际科技媒体视角看中国，以中国视角看全球

## 如何使用

```
使用 pingwest-writer，帮我用品玩风格写一篇关于 [科技产品/趋势/现象] 的观察文章
```
